1) Разархивируйте файл fidoip_win-1.0.5_3.zip в корневую директорию любого диска.
Для того чтобы комплект работал, он должен находиться в корне любого диска в 2-х директориях - usr и home!
(например - d:\usr и d:\home, c:\usr и c:\home и т.д.);

2) Запустите файл \usr\setup_config.vbs, предварительно отредактировав в нём восемь строк:

const yourFullName      = "Vasiliy Pampasov"        	// Тут в кавычках следует указать Ваше имя и фамилию на английском; 
const yourFTNAddress    = "2:5020/828.555"        	// Тут в кавычках следует указать Ваш FTN-адрес, который соощит Вам Ваш босс; 
const yourStationName   = "MyStation"                   	// Тут в кавычках следует указать название Вашей станции, можете выбрать любое слово; 
const yourLocation      = "Moscow, Russia"            	// Тут в кавычках следует указать Ваш город и страну; 
const yourUplinkName    = "Kirill Temnenkov       	// Тут в кавычках следует указать имя Вашего босса; 
const uplinkFTNAddress  = "2:5020/828"              	// Тут в кавычках следует указать FTN адрес узла Вашего босса; 
const uplinkServerName  = "temnenkov.dyndns.org"	// Тут в кавычках следует указать DNS-адрес узла Вашего босса(hostname); 
const yourPassword      = "12345678"     		// Тут в кавычках следует указать пароль на сессию, который Вам сообщит Ваш босс.

3) Зайдите в директорию \usr\bin и щелкните мышкой на файле rs.bat. После того как вы соединитесь и получите сообщения,  запустите редактор GoldEd-NSF щелкнув мышкой на файле \usr\bin\ge.bat. Написав письма и закройте редактор, запустите снова скрипт rs.bat для отправки ваших сообщений;

4) Для Вашего же удобства создайте ярлыки на Рабочем столе для файлов \usr\bin\rs.bat и \usr\bin\ge.bat;

5) Это всё! Для более подробного описания работы поинткомплекта см. файл fido.rus.win.txt каталоге \usr\doc, либо в корневом каталоге архива fidoip*.tar, либо на сайте http://sourceforge.net/apps/mediawiki/fidoip.
